1. Notion de fonction

Définition : Une fonction est un procédé qui associe à chaque nombre d'un ensemble de départ un unique nombre de l'ensemble d'arrivée.

Si la fonction s'appelle f et qu'on lui donne le nombre x, elle renvoie le nombre f(x), appelé image de x par f.
Analogie — machine à calculer :

Considère une machine qui double le nombre et ajoute 1.
Si j'entre x, la machine renvoie 2x + 1.
x
2x + 1
  • Si x = 3 : f(3) = 2 × 3 + 1 = 7
  • Si x = 0 : f(0) = 2 × 0 + 1 = 1
  • Si x = −2 : f(−2) = 2 × (−2) + 1 = −3
Vocabulaire :
  • L'ensemble des valeurs que peut prendre x s'appelle l'ensemble de définition.
  • x est la variable (ou antécédent) de f(x).
  • f(x) est l'image de x par f.

2. Tableau de valeurs

Méthode : Pour dresser un tableau de valeurs d'une fonction f :
  1. Choisir des valeurs de x.
  2. Calculer f(x) pour chacune.
  3. Organiser dans un tableau.
Exemple : f(x) = 2x + 1. Compléter le tableau de valeurs.

3. Fonction linéaire : f(x) = ax

Définition : Une fonction linéaire est une fonction de la forme :

f(x) = ax

où a est un nombre réel appelé coefficient (ou pente).
Propriétés :
  • La représentation graphique d'une fonction linéaire est une droite passant par l'origine O(0 ; 0).
  • Si a > 0 : la droite est croissante.
  • Si a < 0 : la droite est décroissante.
  • Si a = 0 : la droite est horizontale (f(x) = 0 pour tout x).
Exemple : f(x) = 3x et g(x) = −2x.

f(2) = 6 ; f(−1) = −3 ; g(2) = −4 ; g(0) = 0.

4. Fonction affine : f(x) = ax + b

Définition : Une fonction affine est une fonction de la forme :

f(x) = ax + b

• a est le coefficient directeur (pente) de la droite.
• b est l'ordonnée à l'origine (valeur de f(0)).
Propriété : La représentation graphique d'une fonction affine f(x) = ax + b est une droite :
• qui coupe l'axe des ordonnées au point (0 ; b),
• de pente a (si a > 0, elle monte ; si a < 0, elle descend).
Exemple : Tracer la droite représentant f(x) = 2x − 1.

On cherche deux points :
  • x = 0 : f(0) = −1 → point A(0 ; −1)
  • x = 2 : f(2) = 3 → point B(2 ; 3)
On trace la droite passant par A et B.
Remarque : Une fonction linéaire f(x) = ax est un cas particulier de fonction affine avec b = 0. Sa droite passe par l'origine.
