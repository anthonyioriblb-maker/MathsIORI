Développement et factorisation ✨

Nouveau programme cycle 4 · BO 5 mars 2026

1. Vocabulaire

Définitions :

• Développer une expression, c'est la transformer en une somme (ou différence) de termes, en utilisant la distributivité.

• Factoriser une expression, c'est la transformer en un produit de facteurs.
k(a + b)
→ développer
← factoriser
ka + kb

2. Distributivité simple (rappel de 5ème)

Propriété : Pour tous réels k, a et b :

k(a + b) = k·a + k·b     et     k(a − b) = k·a − k·b
Développer :
3(2x + 5)
= 3×2x + 3×5
= 6x + 15
−2(x − 4)
= −2×x − (−2)×4
= −2x + 8
5(3a − 2b)
= 5×3a − 5×2b
= 15a − 10b
Factoriser (mise en évidence d'un facteur commun) :
6x + 10
= 2×3x + 2×5
= 2(3x + 5)
12a − 8
= 4×3a − 4×2
= 4(3a − 2)
x² − 5x
= x·x − 5·x
= x(x − 5)
Méthode pour factoriser :
  1. Identifier le facteur commun à tous les termes (coefficient et/ou partie littérale).
  2. Le mettre en évidence devant une parenthèse.
  3. Remplir la parenthèse : chaque terme divisé par le facteur commun.
  4. Vérifier en développant.

3. Double distributivité

Propriété (double distributivité) :
Pour développer le produit de deux sommes (a + b)(c + d), on multiplie chaque terme du premier facteur par chaque terme du second :

(a + b)(c + d) = a·c + a·d + b·c + b·d

Astuce : on fait 4 multiplications (chaque terme de gauche × chaque terme de droite).
Exemple 1 : Développer et réduire (x + 3)(x + 5).
(x + 3)(x + 5)
= x·x + x·5 + 3·x + 3·5
= x² + 5x + 3x + 15
= x² + 8x + 15
Exemple 2 : Développer et réduire (2x − 1)(x + 4).
(2x − 1)(x + 4)
= 2x·x + 2x·4 + (−1)·x + (−1)·4
= 2x² + 8x − x − 4
= 2x² + 7x − 4
Exemple 3 : Développer et réduire (3x − 2)(3x + 2).
(3x − 2)(3x + 2)
= 9x² + 6x − 6x − 4
= 9x² − 4
Remarque : (a − b)(a + b) = a² − b² est une identité remarquable (différence de carrés).

4. Identités remarquables

Les trois identités remarquables :

(1) (a + b)² = a² + 2ab + b²
carré d'une somme
(2) (a − b)² = a² − 2ab + b²
carré d'une différence
(3) (a + b)(a − b) = a² − b²
différence de deux carrés
Exemples — développer :
(x + 4)²
= x² + 2·x·4 + 4²
= x² + 8x + 16
(2x − 3)²
= (2x)² − 2·2x·3 + 3²
= 4x² − 12x + 9
(5x + 1)(5x − 1)
= (5x)² − 1²
= 25x² − 1
Exemples — factoriser grâce aux identités :
x² + 6x + 9
= x² + 2·x·3 + 3²
= (x + 3)²
4x² − 25
= (2x)² − 5²
= (2x + 5)(2x − 5)
