Calcul littéral — Développement et réduction

1. Rappel — Distributivité simple

Propriété : Pour tous nombres a, b, k :

k × (a + b) = k×a + k×b

Cette propriété permet de développer (enlever les parenthèses) ou de factoriser (mettre en facteur).
Exemples :
  • 3(x + 5) = 3x + 15
  • −2(4x − 1) = −8x + 2  (attention au signe !)
  • x(x + 7) = x² + 7x
  • Factoriser 6x + 10 = 2(3x + 5)
Réduire une expression : regrouper les termes semblables (même puissance de x).
3x + 5 + 2x − 1 = (3x + 2x) + (5 − 1) = 5x + 4

2. Double distributivité

Propriété : Pour tous nombres a, b, c, d :

(a + b)(c + d) = ac + ad + bc + bd
Méthode — développer (a + b)(c + d) :

Chaque terme du premier facteur multiplie chacun des termes du second.

  1. a × c = ac
  2. a × d = ad
  3. b × c = bc
  4. b × d = bd
  5. On additionne et on réduit.

Astuce mémo : « chaque terme de gauche multiplie chaque terme de droite »

Exemple 1 : Développer et réduire (x + 3)(x + 5).

(x + 3)(x + 5) = x×x + x×5 + 3×x + 3×5 = x² + 5x + 3x + 15 = x² + 8x + 15
Exemple 2 : Développer et réduire (2x − 1)(x + 4).

(2x − 1)(x + 4) = 2x×x + 2x×4 + (−1)×x + (−1)×4 = 2x² + 8x − x − 4 = 2x² + 7x − 4
Exemple 3 : Développer et réduire (x − 2)(3x + 1).

(x − 2)(3x + 1) = x×3x + x×1 + (−2)×3x + (−2)×1 = 3x² + x − 6x − 2 = 3x² − 5x − 2
Attention aux signes ! Quand un terme est négatif, le produit change de signe :
(−1) × (−4) = +4  ✓     (−2) × 3x = −6x  ✓

3. Cas particulier : carré d'une somme

À savoir : (a + b)² = (a + b)(a + b)

On applique la double distributivité :
(a + b)² = a×a + a×b + b×a + b×b = a² + ab + ab + b² = a² + 2ab + b²
Résultat : (a + b)² = a² + 2ab + b²

⚠️ (a + b)² ≠ a² + b² — ne jamais oublier le terme 2ab !
Exemple : (x + 3)² = x² + 2×x×3 + 3² = x² + 6x + 9
