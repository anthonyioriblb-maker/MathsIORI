Fonction linéaire

Proportionnalité et fonction linéaire

Voici un tableau de valeurs indiquant le prix à payer en fonction de la quantité de gazole prise à la station-service :

Quantité de gazole (en L) 123456
Prix (en €) 1,42,84,25,678,4

Le prix est proportionnel à la quantité de gazole. Le coefficient de proportionnalité est 1,4.
La courbe obtenue est une droite passant par l’origine du repère. C’est la représentation graphique d’une fonction linéaire : f(x) = 1,4x.

Définition et représentation graphique

Définition : Étant donné un nombre a, la fonction linéaire f de coefficient a est la fonction qui à tout nombre x associe le nombre ax.
On note : f : x → ax   ou   f(x) = ax
Exemple : Soit la fonction f qui à tout nombre x associe son quadruple.
On a f(x) = 4x. C’est une fonction linéaire de coefficient 4.
Propriété :
  • Un repère étant choisi, la représentation graphique de f : x → ax est une droite passant par l’origine du repère.
  • Réciproquement, toute droite passant par l’origine (sauf l’axe des ordonnées) est la représentation graphique d’une fonction linéaire.
  • Le nombre a est appelé le coefficient directeur de la droite.
Méthode — Tracer la courbe de f(x) = 3x :

Comme f est une fonction linéaire, la courbe est une droite passant par l’origine. Il suffit de calculer les coordonnées d’un deuxième point.

On choisit x = 2 :   f(2) = 3 × 2 = 6.

La courbe passe donc par O(0 ; 0) et par le point A(2 ; 6).
L’équation de la droite est y = 3x. Le coefficient directeur est 3.

Fonctions affines

Définition : Soient a et b deux nombres. On appelle fonction affine une fonction qui à tout nombre x associe ax + b.
On note : f : x → ax + b   ou   f(x) = ax + b
Cas particuliers :
 — Si b = 0, alors f(x) = ax : f est une fonction linéaire.
 — Si a = 0, alors f(x) = b : f est une fonction constante.
Propriétés :
  • La représentation graphique de f : x → ax + b est une droite.
  • Toute droite non parallèle à l’axe des ordonnées est la représentation graphique d’une fonction affine.
  • Le nombre a est le coefficient directeur et b est l’ordonnée à l’origine de la droite.
Méthode — Tracer la courbe de f : x → 2x − 3 :

Comme f est une fonction affine, la courbe est une droite. Il suffit de calculer les coordonnées de deux points.

On choisit x = 0 et x = −2 :

f(0) = 2 × 0 − 3 = −3   (ordonnée à l’origine)
f(−2) = 2 × (−2) − 3 = −4 − 3 = −7

La courbe passe donc par A(0 ; −3) et B(−2 ; −7).

Trouver l’expression algébrique d’une fonction

Proportionnalité des accroissements

Propriété : Soit une fonction affine f : x → ax + b. Soient x₁ et x₂ deux nombres.

L’accroissement des images est proportionnel à l’accroissement des abscisses, le coefficient étant a.

C’est-à-dire :   a = f(x₂) − f(x₁)x₂ − x₁
Remarque : Si a > 0 la droite est croissante ; si a < 0 elle est décroissante.
Exemple : Soit une fonction affine g telle que g(−1) = 2 et g(3) = 8. Déterminer l’expression algébrique de g.

La fonction g est affine, donc de la forme g(x) = ax + b.

Détermination de a :

a = g(−1) − g(3)−1 − 3  =  2 − 8−1 − 3  =  −6−4  =  1,5
Ainsi, g(x) = 1,5x + b.

Détermination de b :

On utilise g(3) = 8 :
1,5 × 3 + b = 8 4,5 + b = 8 b = 8 − 4,5 b = 3,5

Ainsi, g(x) = 1,5x + 3,5.
