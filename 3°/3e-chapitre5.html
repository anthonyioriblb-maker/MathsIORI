Équations

Définitions

Définition : Une équation est une égalité contenant une ou plusieurs inconnues (variables).
Définition : Le degré d'une équation est égal à l'exposant le plus grand de l'inconnue x.
Exemples :

3x + 5 = 12 est une équation du premier degré à une inconnue.

2x2 + 3x − 1 = 0 est une équation du second degré à une inconnue.
Attention : le degré d'une équation peut être « caché » :

x2 + 3x + 1 = x2 + 5 est une équation du premier degré car les termes en x2 s'annulent.

Équation du premier degré à une inconnue

Conservation d'une égalité

Règle 1 : L'égalité est conservée quand on ajoute (ou quand on retranche) un même nombre aux deux membres d'une égalité.
Exemples :

• Si 2x + 5 = 13 alors 2x + 5 − 5 = 13 − 5 donc 2x = 8

• Si 3x − 7 = x + 1 alors 3x − 7 − x = x + 1 − x donc 2x − 7 = 1
Règle 2 : L'égalité est conservée quand on multiplie (ou quand on divise) par un même nombre non nul les deux membres d'une égalité.
Exemples :

• Si 2x = 8 alors 2x2 = 82 donc x = 4 (on a divisé par 2)

• Si x3 = 5 alors x3 × 3 = 5 × 3 donc x = 15 (on a multiplié par 3)

Vérification d'une solution

Méthode : Pour savoir si un nombre est solution d'une équation, on remplace les inconnues par la valeur souhaitée.

Rappel : Il faut vérifier une égalité, donc calculer chaque membre de l'équation séparément.
Exemple 1 : Vérifions si 15 est solution de l'équation 2x + 3 = 29.

Pour x = 15 :
Membre de gauche : 2 × 15 + 3 = 30 + 3 = 33
Membre de droite : 29

Comme 33 ≠ 29, 15 n'est pas une solution de l'équation.
Exemple 2 : Vérifions si 13 est solution de l'équation 2x + 3 = 29.

Pour x = 13 :
Membre de gauche : 2 × 13 + 3 = 26 + 3 = 29
Membre de droite : 29

Comme 29 = 29, 13 est une solution de l'équation.

Résolution d'une équation

Définition : Résoudre une équation, c'est trouver toutes ses solutions.
Propriété : Toute équation du premier degré à une inconnue se ramène à une équation de la forme ax = b.
Méthode de résolution d'une équation :

On veut résoudre l'équation 3x + 7 = x − 5, pour cela, on va se ramener à une équation du type ax = b
Résolution détaillée :

3x + 7 = x − 5

Le x est du mauvais côté. Pour le faire disparaître, on va faire −x dans chaque membre.

3x + 7 − x = x − 5 − x

On réduit l'expression :

2x + 7 = −5

Maintenant, c'est le +7 qu'il faut enlever du membre de gauche. Pour l'enlever, on va faire −7 dans chaque membre.

2x + 7 − 7 = −5 − 7

On réduit l'expression :

2x = −12

On obtient la forme ax = b. La solution est x = ba

x = −122 = −6

La solution de cette équation est −6.

Équation produit nul

Propriété : Un produit est nul si et seulement si l'un de ses facteurs est nul.

C'est-à-dire si a et b sont deux nombres relatifs (ou deux expressions) et si a × b = 0 alors a = 0 ou b = 0.
Exemple : Résoudre l'équation suivante : (2x − 4)(3x + 6) = 0

(2x − 4)(3x + 6) = 0

Un produit est nul si et seulement si l'un de ses facteurs est nul.

2x − 4 = 0      ou      3x + 6 = 0

2x = 4      ou      3x = −6

x = 42      ou      x = −63

x = 2      ou      x = −2

L'équation a deux solutions : 2 et −2.

Équation x2 = a

Propriété : Soit a un nombre relatif.

• Si a < 0, alors l'équation x2 = a n'admet aucune solution.

• Si a = 0, alors l'équation x2 = a, admet une unique solution, 0.

• Si a > 0, alors l'équation x2 = a admet deux solutions, a et −a.
Exemples :

• x2 = −9
L'équation n'admet pas de solution.
• x2 = 0
x = 0
• x2 = 9
x = 9 ou x = −9
x = 3 ou x = −3
Les deux solutions sont 3 et −3.
• x2 = 7
x = 7 ou x = −7
Les deux solutions sont 7 et −7.
