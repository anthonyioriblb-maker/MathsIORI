1. Qu'est-ce qu'une équation ?

Définition : Une équation est une égalité qui contient une ou plusieurs inconnues (représentées par des lettres).

Résoudre une équation, c'est trouver la valeur (ou les valeurs) de l'inconnue qui rend l'égalité vraie.

Cette valeur s'appelle la solution de l'équation.
Exemples d'équations :
x + 5 = 12
L'inconnue est x
3a = 15
L'inconnue est a
n − 4 = 7
L'inconnue est n

2. Vérifier une solution

Méthode — vérification : Pour vérifier qu'une valeur est solution d'une équation, on la substitue à l'inconnue et on vérifie que l'égalité est vraie.
Exemple : L'équation est x + 5 = 12. Est-ce que x = 7 est une solution ?

On remplace x par 7 :
7 + 5 = 12 ✓
L'égalité est vraie : x = 7 est bien la solution.
Contre-exemple : L'équation est 3a = 15. Est-ce que a = 4 est une solution ?

On remplace a par 4 :
3 × 4 = 12 ≠ 15 ✗
L'égalité est fausse : a = 4 n'est pas la solution.

3. Résoudre x + b = c

Propriété : Pour résoudre x + b = c, on soustrait b des deux membres :

x + b = c   ⟺   x = c − b

Méthode arithmétique : l'inconnue est le nombre auquel on a ajouté b pour obtenir c. Il suffit donc de calculer c − b.
Méthode en 3 étapes :
1
Repérer b et c dans l'équation x + b = c.
2
Calculer x = c − b.
3
Vérifier en remplaçant x dans l'équation.
Exemple 1 : Résoudre x + 5 = 12.
x + 5 = 12
x = 12 − 5
x = 7

Vérification : 7 + 5 = 12 ✓
Exemple 2 : Résoudre n + 8 = 3.
n + 8 = 3
n = 3 − 8
n = −5

Vérification : −5 + 8 = 3 ✓
Exemple 3 : Résoudre t − 4 = 11.
t − 4 = 11
(on peut écrire : t + (−4) = 11, donc t = 11 − (−4) = 11 + 4)
t = 15

Vérification : 15 − 4 = 11 ✓

4. Résoudre ax = c

Propriété : Pour résoudre ax = c (avec a ≠ 0), on divise les deux membres par a :

ax = c   ⟺   x = c ÷ a

Méthode arithmétique : l'inconnue est le nombre par lequel on a multiplié a pour obtenir c. Il suffit donc de calculer c ÷ a.
Méthode en 3 étapes :
1
Repérer a et c dans l'équation ax = c.
2
Calculer x = c ÷ a.
3
Vérifier en remplaçant x dans l'équation.
Exemple 1 : Résoudre 3x = 15.
3x = 15
x = 15 ÷ 3
x = 5

Vérification : 3 × 5 = 15 ✓
Exemple 2 : Résoudre 4a = 22.
4a = 22
a = 22 ÷ 4
a = 5,5

Vérification : 4 × 5,5 = 22 ✓
Exemple 3 (résultat fractionnaire) : Résoudre 7x = 3.
7x = 3
x = 3 ÷ 7 = 37

Vérification : 7 × 37 = 3 ✓
