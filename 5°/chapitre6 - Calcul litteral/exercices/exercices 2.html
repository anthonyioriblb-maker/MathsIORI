Partie 2 — Développer et réduire
Développe puis réduis si possible les expressions suivantes.
a) 3(x + 5)
b) 4(2a + 3)
c) 5(3x + 1) − 2x
d) x(x + 4) + 2x
e) 4(2x + 3) − 5(x + 1)
f) a(a + 3) + 2(a + 1)
g) 2(3a + 5) − 4a + 3
h) x(x² + 2) − 3x
a) 3(x + 5) = 3×x + 3×5 = 3x + 15
b) 4(2a + 3) = 4×2a + 4×3 = 8a + 12
c) 5(3x + 1) − 2x = 5×3x + 5×1 − 2x = 15x + 5 − 2x = 13x + 5
d) x(x + 4) + 2x = x×x + x×4 + 2x = x² + 4x + 2x = x² + 6x
e) 4(2x + 3) − 5(x + 1) = 4×2x + 4×3 − 5×x − 5×1 = 8x + 12 − 5x − 5 = 3x + 7
f) a(a + 3) + 2(a + 1) = a×a + a×3 + 2×a + 2×1 = a² + 3a + 2a + 2 = a² + 5a + 2
g) 2(3a + 5) − 4a + 3 = 2×3a + 2×5 − 4a + 3 = 6a + 10 − 4a + 3 = 2a + 13
h) x(x² + 2) − 3x = x×x² + x×2 − 3x = x³ + 2x − 3x = x³ − x
